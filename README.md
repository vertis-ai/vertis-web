# Vertis Web 2

Vertis Web application built with TanStack Start.

## Getting Started

To run this application:

```bash
pnpm install
pnpm dev
```

## Development

Start the development server:

```bash
pnpm dev
```

The application will be available at `http://localhost:3000`.

## Building For Production

To build this application for production:

```bash
pnpm build
```

## Testing

This project uses [Vitest](https://vitest.dev/) for testing. You can run the tests with:

```bash
pnpm test
```

## Styling

This project uses [Tailwind CSS](https://tailwindcss.com/) for styling.

## Linting & Formatting

This project uses [Biome](https://biomejs.dev/) for linting and formatting. The following scripts are available:

```bash
pnpm lint
pnpm format
pnpm check
```

## Type Checking

Run TypeScript type checking:

```bash
pnpm typecheck
```

## Shadcn

Add components using the latest version of [Shadcn](https://ui.shadcn.com/):

```bash
pnpx shadcn@latest add button
```

## Built With

This application is built with [TanStack Start](https://tanstack.com/start), a full-stack React framework, and leverages the following TanStack ecosystem packages:

- **[TanStack Router](https://tanstack.com/router)** - File-based routing with SSR support
- **[TanStack Query](https://tanstack.com/query)** - Server state management and data fetching
- **[TanStack Table](https://tanstack.com/table)** - Headless table library
- **[TanStack Store](https://tanstack.com/store)** - Client-side state management
- **[TanStack Form](https://tanstack.com/form)** - Form state management
- **[TanStack Devtools](https://tanstack.com/devtools)** - Development tools for debugging

For detailed documentation on using these tools, see the [cursor rules](.cursor/rules/web-2-general.mdc) or the official [TanStack documentation](https://tanstack.com).
